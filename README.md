# Homebrew Tap for leehosu

## 사용 방법

```bash
# Tap 추가
brew tap leehosu/tap

# commitgen 설치
brew install commitgen
```

## 사용 가능한 Formula

- **commitgen**: AI-powered Git commit message generator
  - OpenAI GPT & Anthropic Claude 지원
  - Conventional Commits 형식
  - 인터랙티브 UI
